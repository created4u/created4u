云计算

定义

云计算就是通过互联网按需提供 IT 资源并采用按需付费定价模式。

部署模式

  • 云端部署
    • 在云中运行应用程序的所有组件
    • 将现有应用程序迁移到云
    • 在云中设计和构建新的应用程序
  • 本地部署/私有云部署
    • 利用虚拟化工具和资源管理工具部署资源
    • 利用应用程序和虚拟化技术提高资源利用率
  • 混合部署
    • 将基于云的资源与本地基础设施连接
    • 将基于云的资源与传统 IT 应用程序集成

优势

  1. 将前期费用变成可变支出

    前期费用是指您在使用数据中心、物理服务器和其他资源之前需要为其投入的费用。可变支出意味着您只需为消耗的计算资源付费，而不需要在知道自己会如何使用数据中心和服务器之前进行大量投资。

  2. 无需在数据中心运行和维护方面投入资金

  3. 无需猜测容量

  4. 实现规模经济效益

  5. 提高速度和敏捷性

  6. 几分钟内实现全球化部署

EC2

工作原理

  1. 启动

    为实例选择具有基本配置的模板，包括操作系统，应用程序服务器或应用程序。还要选择实例类型，即实例的具体硬件配置。

  2. 连接

  3. 使用

实例类型

  1. 通用型

    实现计算，内存和网络资源的均衡。

    • 应用程序服务器
    • 游戏服务器
    • 企业应用程序的后端服务器
    • 中小型数据库

  2. 计算优化型

    适合需要使用高性能处理器的计算密集型应用程序。

    • 高性能 Web 服务器
    • 计算密集型应用程序服务器
    • 专用游戏服务器
    • 成组处理多个事务的批处理工作负载

  3. 内存优化型

    提高用于处理内存中的大型数据集的工作负载的性能。

    • 高性能数据库

  4. 加速计算型

    使用硬件加速器或协同处理器来执行一些功能。

    • 浮点数计算
    • 图形处理
    • 数据模式匹配

  5. 存储优化型

    适用于需要对本地存储上的大型数据集进行高速连续读写访问的工作负载。

    • 分布式文件系统
    • 数据仓库应用程序
    • 高频在线事务处理系统

可扩展性

可扩展性是指仅从需要的资源开始，并且设计架构以便自动扩展和缩减，从而响应不断变化的需求。
